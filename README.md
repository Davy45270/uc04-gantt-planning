# UC04 Gantt & Planning

## Expression de besoin fonctionnelle
- **Objectifs** : planifier via Gantt, visualiser chemin critique, gérer jalons et baselines.
- **Acteurs** : chefs de projet, PMO, sponsors.
- **Flux clés** : saisie tâches/dépendances → calcul planning → baselining → suivi avancement → replanification.
- **Exceptions** : glissements majeurs; re‑baselining validé en comité.
- **Critères d’acceptation** : Gantt interactif, dépendances typées, export planning, historique versions.

## Analyse ROI
- **Gains** : meilleure maîtrise des délais, anticipation des risques, communication planning claire.
- **Coûts** : paramétrage des calendriers, formation à la planification.
- **KPI cibles** : variance de planning, taux de jalons tenus, nombre de re‑baselines.
